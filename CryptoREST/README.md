# CryptoREST #
A REST API that stores historical crypto data. Use this API to backtest 
trading algorithms and bots. 

## Crypto ##
```json
{
    "id": 1000,
    "symbol": "BTC/USD",
    "datetime": "01-01-2024:12:00:00Z",
    "o": 60000,
    "h": 60000,
    "l": 60000,
    "c": 60000
}
```
